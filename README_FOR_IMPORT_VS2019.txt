#НАЗВАНИЕ_ПРОЕКТА - название вашего проекта

Чтобы импортировать библиотеку в Visual Studio 2018 - 2019 неободимо:

1. Распаковать этот архив в рабочую папку вашего проекта
	Рабочая папка вашего проека обычно находиться по адресу C:\Users\#ПОЛЬЗОВАТЕЛЬ\source\repos]#НАЗВАНИЕ_ПРОЕКТА\#НАЗВАНИЕ_ПРОЕКТА
	
2. Открыть свойста вашего проекта в VS
	ПКМ в "Обозреватель решений" по №НАЗВАНИЕ_ПРОЕКТА -> Свойства
	
3. Открыть вкладку "Компоновщик" -> "Ввод"

4. Нажать на стрелочку рядом с полем "Дополнительные зависимости" -> нажать "<Изменить...>"

5. В диалоге "Дополнительные зависимости" в первое текстовое поле вписать "kniitLibrary.lib" (без кавычек)

6. ОК

7. Применить -> ОК

8. Запустить код с использованием библиотеки
